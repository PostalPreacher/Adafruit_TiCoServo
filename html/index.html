Adafruit TiCoServo
Adafruit Timer/Counter Servo Library.

Introduction

This is documentation for the Adafruit_TiCoServo library: uses Timer/Counter 1 (or 3,4,5) to allow use of NeoPixels and servos in the same project (with lots of caveats – see the examples for further explanation). This is similar in ways to the early (pre-0016) Arduino Servo library, but updated for new boards. For AVR & ATtiny only.

Adafruit invests time and resources providing this open source code, please support Adafruit and open-source hardware by purchasing products from Adafruit!
